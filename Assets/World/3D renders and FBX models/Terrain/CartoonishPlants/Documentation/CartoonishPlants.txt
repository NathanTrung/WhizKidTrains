CartoonishPlants Pack  is a low poly set with hand painted textures. 
 
 Inside can be found  in *Fbx and *PNG format the following items: 

 - 3 Types of tree logs;
 - 2 different color cartoonish  pine trees;
 - 2 types of mushrooms;
 - 2 diferent flowers;
 - 1 reed plant;
 - 2 grass meshes;
  - one diffuse 1024x1024 *png texture ;
  - one normal map 1024x1024*png texture;
  - one diffuse 256x256 terrain grass texture;
  - two diffuse 256x256 different color pine trees -tga texture with alpha;
  - one 526x256 alpha grass *tga texture.
It also contains one modified standard shader with double side option for the vegetation.

Suitable for fantasy and cartoonish environments, optimized for mobiles.
